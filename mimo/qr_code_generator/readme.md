# 6. QR code generator & scanner

- Level: Easy

> The QR code generator project shows you how to build a React app
> that both generates and scans QR codes.

このQRコードジェネレーターのプロジェクトは、どうやってReact Appを作るのかを見せます
QRの作成やQRコードをスキャンする

> You’ll create a multi-page application with Material UI components
> for a polished, professional look.

複数ページのアプリケーションを洗練されて、プロっぽく見えるような
Material UI コンポーネントを使ったマルチページアプリケーションを作ります

> QR code generator & scanner React project built with Mimo
> It’ll help you learn how to work with third-party libraries
> for QR code functionality
> while implementing a clean, intuitive interface with proper routing
> between different app sections.

QRコードジェネレータとスキャナーのReactプロジェクトを作ります
これはどう～するかを学べます　QRコードの機能を持ったサードパーティーのライブラリを使うこと
綺麗で、かつ直感的なインターフェイスを持つ一方で
　異なるアプリのセクションを適切なルーティングしながら

> This project also mimics e-commerce features
> like scanning product tags or linking to checkout pages.

## What you’ll build:

> - QR code generator that converts text to QR images
> - QR code scanner that reads codes using a device camera
> - Clean navigation between app sections
> - Responsive design with Material UI components

QRコードGENはtextをQRイメージに変換する
QRスキャナーは、デバイスのカメラを使ってコードを読み込めます
アプリのセクションをいい感じにナビゲートします
Material UIコンポーネンツを使ったResponsive Design

## Prerequisites & tech stack:

> React fundamentals
> Material UI for components
> QR code libraries (qrcode.react and react-qr-reader)
